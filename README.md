# Easy Cookies Importer

## Installation on Firefox

1. Download this repository and extract it if necessary.
2. Open Firefox and enter `about:debugging#/runtime/this-firefox` in the address bar.
3. Click **Load Temporary Add-on**.
4. Open the extension folder and select the `manifest.json` file.
5. The **Easy Cookies Importer** extension will appear in the Firefox toolbar.

> Firefox removes temporary extensions after every restart. Repeat these steps to load it again.
